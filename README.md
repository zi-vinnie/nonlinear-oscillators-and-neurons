# Nonlinear Oscillators and Neurons

## Setup

### 1. Clone the repo

```bash
git clone https://github.com/zi-vinnie/nonlinear-oscillators-and-neurons.git
cd nonlinear-oscillators-and-neurons
```

---

### 2. Create virtual environment

**Windows (PowerShell):**

```powershell
py -m venv .venv
```

**Mac / Linux:**

```bash
python3 -m venv .venv
```

---

### 3. Activate virtual environment

**Windows (PowerShell):**

```powershell
.venv\Scripts\activate
```

**Mac / Linux:**

```bash
source .venv/bin/activate
```

---

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

---

### 5. Run the program

```bash
python main.py
```
